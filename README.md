## Setup Instructions

### Backend
1. Go to `backend` folder
2. Run `npm install`
3. Create a `.env` file from `.env.example`
4. Run `npm start`

### Frontend
1. Go to `frontend` folder
2. Run `npm install`
3. Run `npm run dev`

### Deployment
- Frontend: Vercel
- Backend: Render

MongoDB is required. Use a free MongoDB Atlas account.
